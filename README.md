# Text Generation using GPT-2
This project demonstrates text generation using the GPT-2 language model.

GPT-2 is a transformer-based language model capable of generating 
human-like text based on a given prompt.

Technologies Used:
- Python
- Transformers Library
- PyTorch
- GPT-2 Model

Steps:
1. Install required libraries
2. Load GPT-2 model
3. Provide input prompt
4. Generate text output

Example Prompt:
Artificial Intelligence is

The model generates meaningful text based on the prompt.
